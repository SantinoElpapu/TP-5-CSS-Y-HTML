(1)-Utilize la propiedad background-color con un color negro para que cuando el mouse pase por el article el color de lo que contiene el mismo cambie, agregandole un efecto mas llamativo y estetico. Ademas utilize transition para que el hover (funcion de que al pasar mouse realize un efecto o animación),tenga un efecto y extensión del mismo más largo. Como final use transform que con una escala(1.1) para que al momento de que tanto el background y transition funcionen correctamente el hover realiza el paso importante, dar un efecto "agrandar" para que se vea muy bien.
(2)-La diferencia entre el hover y el before es que, hover realiza un efecto o animación de lo que desee agregar el usuario cuando pase el ratón o mouse, por otro lado before sirve para editar una etiqueta y sin modificarla de html le puedas agregar contenido llamativo(flechas,signos,simbolos,etc).
(3)-Primero va hover y después active. Si los invertís, el hover pisa al active y al hacer clic no se nota el efecto.
(4)-selection es un pseudo-elemento porque estiliza solo la parte del texto seleccionada, ejemplo: cambiar color de fondo al seleccionar. En cambio, las pseudo-clases aplican estilos según el estado del elemento, como hover al pasar el mouse o active al hacer clic.
